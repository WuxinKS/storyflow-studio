# Feature Modules

## 目录职责

- `idea/`：创意输入、关键词扩展、项目初始化
- `story/`：世界观、角色、组织、大纲、章节
- `adaptation/`：章节到 scene / shot 的结构化改编
- `reference/`：截图、参考图、视频抽帧分析
- `storyboard/`：镜头列表、分镜编辑、镜头变体
- `assets/`：角色资产、场景资产、风格卡、提示词模板
- `render/`：视频、配音、字幕、导出与任务编排
- `project/`：项目概览、状态、流程、工作区聚合

## 当前策略

先搭骨架，再逐步把每个模块替换成真实业务实现。
