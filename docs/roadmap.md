# Roadmap

## Phase 0 — 规划冻结
- 完整项目计划
- 产品信息架构
- 核心模块定义
- 数据主链定义

## Phase 1 — 项目骨架
- Next.js 工作台初始化
- 主导航与基础页面
- 基础样式与 app shell
- 文档与 Prisma 初版

## Phase 2 — Story Engine
- 世界观、角色、大纲、章节模型落地
- 故事项目初始化
- 长文本创作工作流

## Phase 3 — Adaptation Engine
- 章节 → Scene
- Scene → Shot
- 镜头脚本与结构化输出

## Phase 4 — Reference Vision Engine
- 单图分析
- 多图分析
- 视频抽帧分析
- 风格卡与镜头模板

## Phase 5 — Storyboard Engine
- shot 可视化编辑
- 分镜图生成
- 镜头变体与人工修订

## Phase 6 — Render Engine
- 图像 / 视频 / 配音 / 字幕链路
- 异步任务、重试、进度跟踪
- 成片导出

## Phase 7 — Product Polish
- 成本可视化
- 资产复用
- 项目版本回溯
- 团队协作与模板能力
